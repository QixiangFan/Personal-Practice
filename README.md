# Personal Practice

A collection of personal scripts and notebooks for learning and practicing machine learning and quantitative finance concepts.

## Structure

### Machine Learning

| Topic | Description |
|-------|-------------|
| `Basic Neural Network/` | MNIST digit classification with a basic neural network |
| `NLP/` | NLP projects including text summarization with LLMs |
| `RNN/` | LSTM-based stock price prediction |
| `LLM/` | Mercari price analysis using large language models |
| `AWS SageMaker/` | Model training and inference on AWS SageMaker |

### Quant Finance

| Script | Description |
|--------|-------------|
| `Black_scholes.py` | Black-Scholes option pricing model |
| `CAPM.py` | Capital Asset Pricing Model |
| `MarkowitzModel.py` | Markowitz mean-variance portfolio optimization |
| `OptionpriceMC.py` | Option pricing via Monte Carlo simulation |
| `StockpriceMC.py` | Stock price simulation via Monte Carlo |
| `vaR.py` | Value at Risk (VaR) implementation |
